CoreUI Alert has been created as an extension of Bootstrap Alert. Alert is delivered with some new features, variants, and unique design that matches CoreUI Design System requirements.

For more information please visit our official documentation.
Alerts

Bootstrap alert is prepared for any length of text, as well as an optional close button. For a styling, use one of the required contextual classes (e.g.,.alert-success).
